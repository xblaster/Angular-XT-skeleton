Angular-XT-skeleton
===================

quick Angular bootstrap less and yo angular skeleton starter

Requirements 
===================

 * Node


install
===================
install npm dependencies
 ```
 npm install 
 ```
 install bower dependencies
 ```
 bower install
 ```
 serve ze app for development
 ```
 grunt serve
 ```
 
build website in dist 
 ```
 grunt build
 ```
